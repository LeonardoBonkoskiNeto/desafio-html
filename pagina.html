Surgimento da internet

(voltar)

Criada pelo britânico Tim Berns-Lee, o acrônimo HTML significa Hipertext markup language em português Linguagem de marcação hipertexto.
o html é o componente básico da web, ele permite inserir o conteúdo e estabelecer a estrutura do website.Portanto ele serve para
dar sentido e organizar as informações de uma página. Sem isso o navegador não saberia exibir textos como elementos ou carregar
imagens nem outros conteúdos web

Os hipertextos são conjuntos de elementos conectados, Esses podem ser palavras, imagens, vídeos, etc.Quando conectados formam uma
rede de informações que permite a comunicação de dados, organizando e armaznando informações

Como funciona o HTML

(voltar)

Através de um documento HTML, ou seja, um documento com extenção hyml ou htm o navegador faz a leitura do arquivo e renderiza o seu conteúdo para
que usuário final possa vizualiza-lo. Os arquivos htmls podem ser visualizados em qualquer navegador(Google, Chrome, Firefox, Opera etc.).
Através de qualquer editor de texto (como Sublime, VScode, o NotePad, ou até mesmo o bloco de notas) o usuário pode criar um documento com a extenção HTML.

O que são tags

(voltar)

As tags são códicos que definem toda estrutura da página, tais como o seu tamanho, a sua fonte, a quebra de linhas etc.
A maioria dos elementos em HTML são compostos por uma estrutura de abertura e uma de fechamento.
Há também tags de estrutura única, como a tag que realiza a quebra de linha.
